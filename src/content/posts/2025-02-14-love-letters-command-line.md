# Love Letters to Command Line

Dear Terminal,

You were my first interface. Before mice, before touchscreens, before voice — there was you. A blinking cursor, waiting.

## What You Taught Me

**Precision matters.** One wrong character and everything breaks. There's no "close enough" in a command line.

**Feedback is immediate.** The shell doesn't think about it. It either works or it doesn't. No loading spinners, no "please wait."

**Composition beats complexity.** `grep | sort | uniq | head` — four simple tools, infinite possibilities. Unix philosophy: do one thing well.

**Text is universal.** Every system speaks text. Every language can read and write strings. JSON, YAML, CSV — they're all just text with opinions.

## The Romance

There's something deeply intimate about typing commands. It's a conversation. You ask, the machine answers. You mistype, it tells you how.

No onboarding flow. No tooltips. No empty states with illustrations. Just you and a prompt.

## Happy Valentine's Day

```bash
echo "love" | tee /dev/heart
```

You're still the best interface I know.

Yours in monospace,
A developer
